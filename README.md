## 🚀 Flagship Projects

### [CodeCollab IDE](https://github.com/Neetworm/CollaborativeIDE)
**Real-Time Collaborative Cloud Workspace** A production-grade IDE allowing concurrent developers to write, compile, and sync code in the same room.
* **Performance:** Achieved sub-150ms bidirectional synchronization latency using Socket.IO rooms and in-memory caches.
* **Security:** Engineered a remote code execution engine using Docker sandboxing to isolate compilations (C++, Python, JS).
* **Data Handling:** Managed project structural metadata via MongoDB schemas while offloading file payloads to AWS S3.

### [ElevateU+ Resume Analyzer](https://github.com/Neetworm)
**Automated Parsing & Processing Engine** A full-stack web application designed to integrate frontend interfaces with optimized database backends.
* Designed modular backend routes to handle file intake, processing, and automated scoring workflows.
* Optimized MySQL querying protocols to improve overall database retrieval speed and reliability.

**Asynchronous Content Moderation System** Backend-supported web system for rule-based text evaluation.
* Constructed database schemas ensuring high data consistency for moderation logging.
* Implemented Node.js asynchronous request handling to prevent thread blocking during heavy evaluation tasks.
